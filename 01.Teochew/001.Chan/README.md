### 一、原作序言

```
敬告：
        關於家史概述及各房先代墳墓名稱輩份等，未能查詳。
        皆因年久，由其各房居住分散及工作關係，聯繫甚難。
        知情難找及人力學識等各種因素限制，只能簡單修整，不達其全而成憾事。
        請多諒解，並希各房另行補全相互聯繫。

    順致謙意。

二〇〇七丁亥歲
```
